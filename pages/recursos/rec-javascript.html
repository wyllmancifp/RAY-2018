JavaScript

Introducción y recursos básicos

¿Qué es JavaScript?

Un programa es un trozo de texto (código) escrito por un programador, formando un conjunto de instrucciones que fuerza directamente al ordenador a cumplir su cometido.

JavaScript es un lenguaje de scripting multiplataforma y orientado a objetos, introducido en 1995, usado principalmente para páginas web, compatible con todos los navegadores y que forma un estándar de desarrollo que ahora también se extiende a dispositivos o programas de propósito general multiplataforma.

Fue creado por Brendan Eich, al principio el lenguaje se llamaba Mocha, luego lo renombró como LiveScript hasta que finalmente le cambiaron el nombre a JavaScript. JavaScript no tiene casi nada que ver con el lenguaje de programación Java.

Después de esta aceptación fuera de Netscape, un documento estándar fue escrito para describir la forma en que el lenguaje JavaScript debería escribirse y funcionar para hacer compatible otros programas con JavaScript. Se llamó el estándar ECMAScript, después la Ecma International Organization hizo la estandarización. En la práctica, los términos ECMAScript y JavaScript se pueden usar indistintamente, son dos nombres para el mismo lenguaje.

JavaScript puedes usarlo como lenguaje de programación del lado del cliente (se ejecuta en nuestro ordenador, no en el servidor). Cuando un usuario accede a una página web cualquiera, envía desde su navegador una petición a un servidor web que nos devuelve “el servicio”. Nos brinda una página web que visualizamos inmediatamente en nuestro navegador, es decir, esa petición envía al ordenador del usuario código HTML junto a código JavaScript.

JavaScript no está limitado solo a los navegadores web. Las implementaciones de Javascript en el lado del servidor ha ido ganando adeptos desde el año 2000, siendo NodeJS una de las más populares, usándose así para el desarrollo de aplicaciones web escalables. NodeJS está construido sobre el intérprete o motor de JavaScript de Google Chrome (motor V8).

Ya sea con navegadores o con NodeJS del lado del servidor, no son las únicas plataformas donde se emplea JavaScript. Algunas bases de datos como MongoDB usan JavaScript para sus consultas. En realidad JavaScript puede estar en cualquier plataforma que tenga instalado un intérprete de JavaScript, como dijimos anteriormente, es un lenguaje muy flexible en todos los aspectos.

Sintaxis y conceptos básicos

- JavaScript es case-sensitive (distingue mayúsculas y minúsculas) y utiliza el conjunto de caracteres Unicode.
- Un identificador ha de empezar con una letra, un guión bajo (_) o un símbolo de dólar ($); los valores subsiguientes puede ser números o letras que incluyen tanto desde la "A" hasta la "Z" (mayúsculas) como de la "a" hasta la "z".
- Los identificadores no pueden ser iguales a ninguna de las palabras reservadas del lenguaje.
- No se tienen en cuenta los espacios en blanco y las nuevas líneas
- Los bloques de código se escriben entre llaves { } - Las instrucciones son llamadas Sentencias y son separadas por un punto y coma (;).
- Para comentarios de una sola línea, se empieza ésta con //
- Para comentarios multilínea se empieza con /* y se termina con */
- Para ejecutar código JS en HTML se introduce entre etiquetas "script" o enlazando un archivo externo de código en el atributo "src" de dicha etiqueta.
- Para visualizar textos por pantalla o consola (usado principalmente para depuración y fases de desarrollo) se puede usar console.log("Hola mundo");

									
<script>
	// Este es un comentario de una sola línea

	/*
		Esto es un comentario multilínea
	*/

	{
		// Bloque de código
	}

	console.log("Texto a mostrar en la consola");
</script>

Variables y tipos de datos base

Una variable se utiliza para almacenar y hacer referencia a otro valor en el espacio de memoria del programa.

Declarar variables consiste en definir y de paso informar al sistema de que vas a utilizar una variable. Javascript cuenta con la palabra "var" que utilizaremos cuando queramos declarar una o varias variables. Como es lógico, se utiliza esa palabra para definir la variable antes de utilizarla. Aunque Javascript no nos obligue a declarar explícitamente las variables, es aconsejable (y buena práctica) declararlas antes de utilizarlas. Se puede asignar un valor a la variable cuando se está declarando y e permite declarar varias variables en la misma línea, siempre que se separen por comas.

Se le llama ámbito de las variables al lugar donde estas están disponibles.
- Las variables globales son las que están declaradas en el ámbito más amplio posible. Éstas son accesibles desde cualquier lugar de la página.
- Las variables locales se declaran dentro de un bloque de código de una función. Éstas serán accesibles desde cualquier parte del bloque donde se haya definido, incluyendo bloques anidados.

En las variables podemos guardar distintos tipos de información. Los tipos de datos básicos son booleanos (verdadero o falso), numéricos (enteros o reales) o textos (cadenas de caracteres entre comillas), listas o vectores de los tipos básicos y objetos

var variable002 = 1; // Variable declarada y asignando un tipo numerico
var variable003 = "hola"; // Variable declarada y asignando un tipo texto o "String"

var variable004, variable005;

variable004 = true; // Asignación de un valor booleano a una variable
variable005 = 25; // Asignación de un valor numérico a una variable

Operadores y estructuras de control

Los operadores permiten manipular el valor de las variables, realizar operaciones matemáticas con sus valores y comparar diferentes variables. De esta forma, los operadores permiten a los programas realizar cálculos complejos y tomar decisiones lógicas en función de comparaciones y otros tipos de condiciones.

Los tipos básicos de operadores son:
- Asignaciones (binario): =, +=, -=, *=, /=
- Incremento/Decremento (Unario): ++, --
- Aritméticos (binarios): +, -, *, /, %
- Relacionales (binarios): <, >, <= , >=
- Igualdad, desigualdad: ==, !=
- Condicionales: &&, || 

							
// Ejemplos de los operadores básicos
var numerico001 = 0;

// Asignaciones
numerico001 = 10; // El valor de la variable numerico001 se ha asignado en 10
numerico001 += 9; // Se le asigna a la variable su valor anterior sumandole 9
numerico001 -= 17; // Se le asigna a la variable su valor anterior restandole 17
numerico001 *= 2; // Se le asigna a la variable su valor anterior multiplicandolo por 2
numerico001 /= 2; // Se le asigna a la variable su valor anterior dividiendolo por 2

// Incremento/Decremento
numerico001++; // Se le suma 1 al valor actual de la variable
numerico001--; // Se le resta 1 al valor actual de la variable

// Aritmeticos
numerico001 = 1 + 1; // Suma
numerico001 = 1 - 1; // Resta
numerico001 = 1 * 2; // Multiplicación
numerico001 = 4 / 2; // Division
numerico001 = 3 % 2; // Resto de la division entera

var booleano001 = true;

// Relacionales
booleano001 = 2 > 1; // Mayor que
booleano001 = 1 < 2; // Menor que
booleano001 = 5 >= 5; // Mayo o igual que
booleano001 = 4 <= 5; // Menor o igual que

// Igualdad/Desigualdad
booleano001 = (5 == 5); // Igual a
booleano001 = (3 != 5); // Distinto a
booleano001 = ("hola" == "hola");

// Condicionales
var booleano002 = false;

booleano001 = (true && true); // Operador "y" lógico
booleano001 = (booleano002 || true); // Operador "o" lógico

Las sentencias de bloque son el tipo de sentencias más básicas y se utilizan para agrupar otras sentencias. Suelen estar definidas por una cabecera definitoria y un bloque de codigo definido entre llaves { }. Éstos bloques se usan principalmente para el control del flujo de ejecución del programa.

Sentencias condicionales: se define como un conjunto de comandos que se ejecutan si una condición es verdadera. JavaScript soporta las dos sentencias condicionales base: if...else y switch

Funciones, definición y uso

Una función es una porción de código reusable que puede ser llamada en cualquier momento desde nuestro programa. Se compone de una cabecera definitoria con un identificador y los parámetros a usar entre paréntesis. Seguido de un bloque de código a ejecutar cada vez que se llame a dicha función.

function nombre_funcion(parametros) {
instrucciones;
}
				

// Ejemplos de definición y llamadas a funciones
var variableGlobal001 = "texto global";

// Definición de una función que no usa parámetros y no retorna valor alguno
function holaMundo () {
	console.log("hola mundo");
}

// Llamada a la funcion
holaMundo();

//------------------------------

// Definición de una función que usa dos parámetros de entrada y no retorna valor alguno
// La variable textoAMostrar contendrá el valor de tipo string a mostrar por consola
// La variable numeroVeces se usará como contador de las veces a mostrar el texto anterior
function mostrarTexto (textoAMostrar, numeroVeces) {
	
	for (var i = 1; i < numeroVeces; i++) {
		console.log("El texto a mostrar por consola es: " + textoAMostrar);
	}
}

// Llamadas a la función;
mostrarTexto("hola", 2);
mostrarTexto("feo", 1);
mostrarTexto("HOLA MUNDO", 20);

//------------------------------

// Definición de una función que usa dos parámetros de entrada y retorna un valor numérico
// Las variables primerValor y segundoValor contendran los valores numéricos a sumar
function sumarValores (primerValor, segundoValor) {
	console.log("Sumando " + primerValor + " y " + segundoValor);
	return primerValor + segundoValor;
}

// Llamadas a la función
var variableResultado = sumarValores(2, 2);
variableResultado = sumarValores(1, 2);
